## 运行环境
- PHP > 7.0
- laravel框架支持的数据库

## 使用方法
### 安装
```
git clone https://github.com/kamino-space/CloudRich.git
cd CloudRich
cp .env.example .env
php artisan key:generate
vi .env
php artisan migrate --seed
```
### 后台
yourdomain/admin
